Contributing to WebODF

Requirements

WebODF is a JavaScript library. To develop for it you need a text editor and software that can run JavaScript. Nearly all modern browsers can do this. On top of that, there is support for JavaScript in many other programs. For WebODF development, we rely a lot on unit tests. These run in the browser and in Node.JS and Rhino. They tests should pass in all of these environments.

The main script for running the tests currently runs in Node.JS. This is because this is the fastest command-line runtime.
