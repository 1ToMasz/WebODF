WebODF

About

WebODF is a javascript library that makes it easy to add Open Document Format (ODF) support to your website and to your mobile or desktop application. It uses HTML5 to display ODF documents in the browser.

Getting involved

Demo

There is a demo website for WebODF. It has a large set of testdocuments available. You can explore the website to see how well WebODF can render the various ODF documents.
